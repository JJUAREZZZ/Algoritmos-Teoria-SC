PROYECTO CORTO DE ALGORITMOS Y ESTRUCTURAS DE DATOS
QuickLibrary:

Sistema de Gestión de Préstamos de Libros

1. Introducción
Una biblioteca universitaria necesita organizar sus libros y atender las solicitudes de
préstamo de los estudiantes. Actualmente, la información se registra de manera
desordenada, lo que dificulta localizar libros y respetar el orden de atención de las
solicitudes.

En este proyecto, los estudiantes desarrollarán una aplicación denominada QuickLibrary,
orientada a gestionar un catálogo pequeño de libros y una cola de solicitudes de
préstamo.

El proyecto deberá ser desarrollado en Java, aplicando estructuras de datos
implementadas por los propios estudiantes.
